# Dashboard

## Live version

[Click](https://dashboard-topaz-tau-65.vercel.app/)

## Tech stack
React, styled-components, redux, typescript

## Features:

- Task list
- Displaying current news 
- Displaying cryptocurrency chart (12 hours range)
- Weather forecast
- Current date
- Changing background using images from pixabay 

## How to run

```sh
   npm run install
   npm run dev
```
